AI Agent Testing Page for Spirit Airlines

Welcome to the testing page for the Spirit Airlines AI Agent. Please follow the guidelines below to ensure effective testing:

  • Verify that the AI responds accurately to customer inquiries.
  • Test the AI's ability to handle unexpected user input.
  • Assess response time for typical customer questions.
  • Check the AI's capability to escalate complex issues to human agents.

Testing Scenarios for Spirit Airlines AI Agent

Scenario: Inquiry about Voucher Usage

  • User Story: As a guest, I want to inquire about my voucher so that I can understand how to use it.
  • Context: Guest has received a voucher and needs information on its usage.
  • Initial User Input: I have a voucher and I need to know how to use it.
  • Potential Escalation Scenarios: If the guest has trouble redeeming the voucher or if the voucher is expired or invalid.
  • Expected Resolution: The guest is provided with detailed information on how to redeem the voucher, including applicable flights, booking process, and any terms and conditions.
  • Acceptance Criteria: The guest receives clear instructions on how to use the voucher, including any restrictions or expiration dates.
